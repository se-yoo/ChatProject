# ChatProject
자바 채팅 상담 프로그램

## 기능 ##
### 1. 공통 ###
- 로그인

### 2. 관리자 ###
- 채팅 목록 확인
- 사용자와의 채팅

### 3. 사용자 ###
- 관리자와의 채팅(사용자 당 한명의 관리자와 매칭되도록 함)
